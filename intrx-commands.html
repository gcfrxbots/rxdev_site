INTRX  COMMANDS

If you followed the setup guide, IntRX is ready to use. But to actually make use of it, you have to set up your interaction commands.

CREATING  YOUR  COMMANDS

Open InteractConfig.xlsx in the Config folder. This spreadsheet is where you'll set up your commands for various games.
The "Command" column is what users will type into chat (starting with !) to execute the command. The "Cooldown" column is the number of seconds after a command is executed before it can be executed again. The "Command to Execute" column is the command that will be executed in-game. And if you want to disable a command, simply type "Disable" into the "Disable" column.
And at the bottom of the screen, you will see multiple tabs, for all the different sheets— each game gets its own sheet. So if you want to set up commands for Skyrim, you would switch to the "Skyrim" sheet. Commands set up in a game-specific tab will only work in that specific game. The "Global" sheet is for creating commands that will always be active, regardless of what you're playing.
The "Global" sheet is a bit different from game-specific sheets. It has an "Active Window" column, which is where you put the name of the program window that you want the command to be for. And instead of a "Command to Execute" column, it has a "File Name to Run" column, where you type the name of a custom AHK script you've created and placed in the UserScripts folder. Read the guide on Creating Your Own Scripts for more info.
Note that you must use AutoHotkey to compile your scripts into an .exe file to use them. If you have AutoHotkey installed, this is as simple as right-clicking your script and selecting "Compile Script" in the context menu. So if your script file was called SpawnBear.ahk, compiling it would turn it into SpawnBear.exe. And to create a command which uses this script, simply type "SpawnBear" in the "File Name to Run" column of InteractConfig.xlsx's Global sheet.
Every game has its own method of inputting commands— you likely understand what that is if you've played the game. For example, Skyrim commands are inputted via the in-game console, and Minecraft commands are done in the in-game chat. Commands can be set up to do things like spawn enemies, add items to the player's inventory, etc. If you're unsure how to use cheats or commands in your game of choice, just Google search the name of your game followed by "console commands" or "cheat codes".
It is worth mentioning that the spreadsheet only cares about what you type. So if you'd like to customize and/or organize your spreadsheet, feel free to do things like change fonts, recolor text or cells, adjust column width, etc. Just don't merge cells or change the text itself.

ADDING  SUPPORT  FOR  YOUR  OWN  GAMES

IntRX supports a few games by default, but odds are, you will want interaction commands for a different game. Well you're in luck, that's what the "Global" sheet in InteractConfig.xlsx is for. There is a whole page on this site dedicated to teaching you about Creating Your Own Scripts for any game you want using AutoHotkey. But don't let it intimidate you, it is quite simple.
