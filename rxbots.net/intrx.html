IntRX (pronounced like "Interax") is a unique Twitch bot which allows viewers to directly interact with the streamer's game using chat commands. Several games are supported by default, such as Minecraft, Skyrim, Fallout 4, and Subnautica, but IntRX also allows its users to add support for any game they want with relative ease. Click the download button to get the latest version, then hover over the IntRX button in the top right corner of the page to navigate to the setup guide!
